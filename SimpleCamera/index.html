Use your mobile device's camera

  • visit this page with a mobile device
  • click the 'take photo' button
  • select the device's camera as the source
  • take a photo
  • the photo should display above
  • if you visit from the desktop browser, this component will open the file selection dialog and not the web camera
  • Open source! https://github.com/rendall/simple-components/tree/master/SimpleCamera
